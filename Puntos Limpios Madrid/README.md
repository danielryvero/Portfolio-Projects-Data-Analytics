# Interactive Dashboard Report Power BI

Dasboard created in Power BI with data extracted from the [European Data](https://data.europa.eu/data/datasets/https-datos-madrid-es-egob-catalogo-200284-0-puntos-limpios-fijos?locale=en) webpage. Data from a [json file](https://datos.madrid.es/portal/site/egob/menuitem.ac61933d6ee3c31cae77ae7784f1a5a0/?vgnextoid=00149033f2201410VgnVCM100000171f5a0aRCRD&format=json&file=0&filename=200284-0-puntos-limpios-fijos&mgmtid=2bb427e0cb503410VgnVCM1000000b205a0aRCRD&preview=full) is extracted into Power Query and Visualized in the Power BI Desktop. 

Dashboard is available for [download](https://drive.google.com/drive/folders/1txoUIO80g6ekvWErHUfou5TA2qCdM2ZS?usp=sharing) and engange with its interactivity using the Desktop version of the tool.

JSON file in GitHub is [here](/Puntos%20Limpios%20Madrid/200284-0-puntos-limpios-fijos.json).
